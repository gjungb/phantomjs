`addCookie([Cookie](#cookie))` {Boolean}

**Introduced:** PhantomJS 1.7

Add a [Cookie](#cookie) to the CookieJar.  Returns `true` if successfully added, otherwise `false`.

## Examples

```javascript
phantom.addCookie({
  'name': 'Added-Cookie-Name',
  'value': 'Added-Cookie-Value',
  'domain': '.google.com'
});
```
